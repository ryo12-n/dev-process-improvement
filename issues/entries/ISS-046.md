## 課題詳細

L2-worker サブエージェントの実機検証（T-002）で、エージェント定義の `tools: ["Read", "Write", "Edit", "Glob", "Grep", "Bash"]` に含まれていないツール（WebSearch, WebFetch, TodoWrite, NotebookEdit, Skill）が全て利用可能だった。

`tools` フィールドは Claude Code CLI の Task ツール起動時にのみ適用される制限であり、Claude Agent SDK 経由の起動では制限が適用されない。エージェント定義の `tools` フィールドは「意図の文書化」としての価値はあるが、強制力を持つ制限ではない点を認識する必要がある。

## 対応方針（案）

[案] エージェント定義の `tools` フィールドは「最小権限の意図を文書化するもの」として維持する。SDK 環境での制限が必要な場合は、SDK レベルでの制御（canUseTool コールバック等）を検討する。
