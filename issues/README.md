# issues/ ディレクトリ

## 用途

課題詳細の管理ディレクトリ。`プロセス改善_課題管理.csv`（全体俯瞰ハブ）と対になる1課題1ファイル形式で、CSV の1行では書ききれない背景・経緯・前提条件・参照資料などの詳細情報を管理する。

## 命名規則

- ファイル名: `ISS-XXX.md`（CSV の ID と一致させる）
- テンプレート: `_template.md` をコピーして新規作成する
- 例: `ISS-038.md`, `ISS-039.md`

## CSV との関係

| 管理対象 | CSV | issues/ |
|---------|-----|---------|
| 全体一覧・ステータス管理 | 主 | - |
| 課題の詳細情報 | 要約（3行以内） | 詳細（制限なし） |
| リンク | `詳細ファイル` 列で参照 | - |

CSV の `詳細ファイル` 列に `issues/ISS-XXX.md` のパスを記入することで、CSV から詳細ファイルへのナビゲーションを確保する。

## 作成タイミング

L2（実施/評価）が `07_issues.md` の課題を CSV へ転記する際に、同時に `issues/ISS-XXX.md` を作成する。
