## 発見された課題

### 課題1: MC-005 の未実行判定における inbox 現存ファイル照合の不完全性

backlog-maintenance-session #1 の inbox ルーティングが「未実行」と判定されているが、`inbox/session-flow-checklist-precision.md` が現存し、かつファイル内で発見元として当該 gate_review の必須把握事項 #1 を明記している。同様に #2 も `inbox/triage-standard-policy-table-width.md` として現存。

考えられる原因: inbox ファイルの検索時にキーワードマッチの精度が不足していた可能性。gate_review の知見タイトル（「新セッションタイプ追加時の連動更新箇所がチェックリスト以上に多い」）と inbox ファイル名（`session-flow-checklist-precision.md`）の間に直接的な語彙の一致がないため、マッチングが失敗した可能性がある。

### 課題2: 他の未実行判定にも同様の見落としがある可能性

2件の誤判定が確認されたことで、残り12件の未実行判定にも同様の見落としがある可能性を排除できない。特に inbox ファイル名と gate_review の知見タイトルが異なるケースでは、キーワードマッチだけでは検出が困難。
