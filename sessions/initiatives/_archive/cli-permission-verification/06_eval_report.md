## 発見された課題・改善提案

### EVL-001: Write deny パターンの `*.pem`/`*.key` 欠落

- **内容**: settings.json の deny リストで、Read/Edit には `**/*.pem`/`**/*.key` の deny があるが、Write にはない。ISS-048 で「Read/Write/Edit 網羅性チェック」の重要性を指摘しつつ、同じ修正作業内で Write の `*.pem`/`*.key` を追加し忘れている。
- **影響**: `.pem`/`.key` ファイルの新規作成（Write）は deny されない。実害は低い（作成よりも読み取り・変更の方がリスクが高い）が、対称性の観点から修正すべき。
- **推奨対応**: `Write(**/*.pem)`, `Write(**/*.key)` を deny リストに追加する。
