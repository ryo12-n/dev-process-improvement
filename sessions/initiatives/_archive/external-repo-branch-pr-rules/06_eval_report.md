## 発見された課題・改善提案

### EVL-001: `gh -C <path>` が gh CLI v2.87.3 でサポートされていない

**影響箇所**:
1. `.claude/rules/session-start-branch-cleanup.md` ステップ6の `gh -C <path> pr list` / `gh -C <path> pr create`
2. `.claude/skills/l1-manager/SKILL.md` クローズ手順ステップ7の `gh -C <path> pr create`

**影響度**: 中 — `git -C <path>` は正常に動作するため、ブランチ整理の大半（ブランチ確認・push・checkout）は問題なし。PR 確認・作成のステップのみ影響。

**推奨修正**: `gh -C <path>` の記載を `cd <path> && gh pr list/create` に変更する。`git -C` との混在になるが、`gh` 側に `-C` オプションがない以上フォールバックが必要。

→ `07_issues.md` に EVL-001 として起票済み
