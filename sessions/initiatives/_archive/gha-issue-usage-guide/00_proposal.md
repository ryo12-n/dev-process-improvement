# 改善施策提案: GHA Issue コメントにユーザー向け使い方ガイドを表示

## 背景・課題

GHA Initiative フローでは Issue コメント経由で `@ai-task /start` 等のコマンドを使うが、メタデータコメント投稿時と backlog-to-issue 起票時にユーザーへの使い方案内がない。初めて使うユーザーや久しぶりに使うユーザーが次のアクションを把握しにくい。

## 目標

- メタデータコメントにコマンドリファレンスを追加（`<details>` 折り畳み）
- backlog-to-issue の Issue 本文に `/start` の案内を追加

## スコープ
### やること
- `initiative-wallbash.yml` のメタデータコメントにコマンドリファレンス追加
- `backlog-to-issue.yml` の Issue 本文に使い方案内追加
- `backlog-to-issue.yml` の `${{ }}` 直接展開をセキュリティルール準拠に修正

### やらないこと
- 既存の壁打ちサマリーの「次のステップ」案内の変更
