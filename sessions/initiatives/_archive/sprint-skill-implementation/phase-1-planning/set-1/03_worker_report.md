## 所感・次フェーズへの申し送り

- Wave 2 の3タスク（T-002/T-003/T-004）は実施計画書の仕様をそのまま SKILL.md に転記する作業が中心だが、指摘B（起動キーワード区別）の反映方法がワーカーの判断に委ねられる部分がある。L1 がタスク作成時に指摘B の具体的な反映方針（例: 各 SKILL.md の起動パターンセクションにユーザー発話例を追加する等）を明示することを推奨する
- `sprint_template` の参照箇所（23ファイル）のうち、アクティブファイルへの参照更新が必要かどうかの判断は T-001 ワーカーに委任する。`inbox/sprint-skill-implementation 実施計画書.md` 内の参照は施策完了後に inbox ファイル自体が削除されるため更新不要
