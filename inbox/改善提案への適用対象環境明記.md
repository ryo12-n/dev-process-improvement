# 改善提案への適用対象環境明記

## 気づき

L2-worker が作成する改善提案（02_tasks.md の T-005 相当）において、「どの環境を前提とした提案か」を必ず明記すべき。

## 背景

`subagent-permission-model` 施策で CLI 環境と SDK 環境で挙動が異なることが判明したが、改善提案の前提環境が暗黙的になっている箇所があった。

## 提案

タスク設計時（02_tasks.md 作成時）に「提案には適用対象環境を明記する」ことを改善提案タスクの完了条件に含めることを検討。

## 発見元

- 施策: `sessions/initiatives/subagent-permission-model/`
- 評価レポート: `06_eval_report.md` ルール化候補 #1
