# チェーン型 Skill プロンプト設計ガイドラインの策定提案

## 背景

ai-dev-chain-gen-req 施策で、gen-req Skill のプロンプト実装パターンを確立した。「ステップ番号付き処理フロー + スキーマ/テンプレートのインライン埋め込み」の構造が効果的であり、後続の gen-design 等でも再利用可能。

## 提案

gen-req で確立したプロンプト設計パターンを `.claude/rules/` のガイドラインとして策定し、gen-design 以降のチェーン型 Skill 実装で一貫して適用する。

### パターンの要素

1. 7ステップ構成: パス取得→バリデーション→上書き確認→抽出/変換→JSON生成→MD生成→報告
2. スキーマ定義を JSON コードブロックでインライン埋め込み
3. テンプレート定義を Markdown コードブロックでインライン埋め込み
4. 各ステップに判定基準とエラーハンドリングを明示
