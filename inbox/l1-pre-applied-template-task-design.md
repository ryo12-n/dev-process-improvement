# テンプレート変更施策での L1 事前適用の明示化

## 気づき

L1 がディスパッチ指示にテンプレートの新構造を含めつつ、テンプレート自体も先行適用するケースがある。この場合、L2 のタスクが「確認のみ」で完了し、タスク設計時の想定と実作業に乖離が生じる。

## 提案

タスク設計時に「L1 事前適用済み」と「L2 による新規作成」を明示的に区別し、L2 の作業見積もりの精度を向上させること。

- 作業レポート: `04_work_report.md` ルール化候補 #1
- 評価レポート: `06_eval_report.md` 参考情報 #1
