# 費用対効果評価フレームワークの標準化

## 気づき

ai-automation-tool-review 施策で使用した費用対効果の評価フレームワーク（頻度x手動コスト/実装難易度）は、自動化に限らず「やるべきか」の優先順位付けに汎用的に使える。

## 具体的な内容

- **評価軸**: 頻度（毎セッション=3 / 毎施策=2 / 不定期=1）、手動コスト（高=3 / 中=2 / 低=1）、実装難易度（高=3 / 中=2 / 低=1）
- **計算式**: 優先度スコア = (頻度 x 手動コスト) / 実装難易度
- この定義をテンプレートに組み込むことで、将来の施策で優先度の定量評価を迅速に行える

## 発見元

`sessions/initiatives/ai-automation-tool-review/04_work_report.md` T-003

## 提案

`templates/` または `docs/` に評価フレームワークのテンプレートを追加する。
