# 統合施策の優先度スコア算出方法の定義

## 気づき

オートメーション走査の AT-004（backlog 起票提案）で、複数の backlog エントリを統合する施策の優先度スコア算出方法が未定義であることが判明。ワーカーは「推定 4.5」と記載したが、計算根拠が曖昧。

## 改善提案

automation-candidates の運用ガイドラインに統合施策のスコア算出方法を定義する。候補:
- **合算方式**: 元エントリのスコアを合計（重複分を控除）
- **最大値方式**: 元エントリの最大スコアを採用
- **加重平均方式**: 実装共有度に応じた加重平均
