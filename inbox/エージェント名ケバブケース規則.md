# エージェント名のケバブケース規則の明文化

## 気づき

Claude Code agents の `name` フィールドは小文字・ハイフンのみを許容する。roles/ 時代のスネークケース（`feature_builder`）からケバブケース（`feature-builder`）に変換が必要。`role-format-guide` にこの命名規則を明記することで、新規エージェント作成時の迷いを防止できる。
