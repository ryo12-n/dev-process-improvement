# 再評価テーブルのカウント定義の明確化

## 気づき

オートメーション走査の AT-002（既存候補再評価）で、「変更提案あり」のカウントに「検討したが据え置き」を含めるか否かの基準が曖昧であることが判明。evaluator が軽微な記載整合性問題として指摘。

## 改善提案

automation-worker のスキャン計画テンプレートまたはワーカー指示に、以下のカウント定義を明記する:
- 「変更提案あり」: 優先度スコア・手動コスト・ステータス・自動化方針のいずれかに変更がある候補
- 「据え置き（検討済み）」: 検討した結果変更なしと判断した候補（カウントには含めない）
