# スキル修正時の担当ファイルテーブル網羅性チェック

## 種別
ルール改善の種

## 内容
スキルファイル（`.claude/skills/*/SKILL.md`）の作業フロー（Step N）を修正する際、サブステップで言及される操作対象ファイルが「担当ファイル」テーブルに全て含まれていることを確認するチェックを、L2-worker のタスク完了条件に明示的に含めるべき。

## 背景
automation-action-step 施策で、Step 7 サブステップに `issues/entries/ISS-XXX.md` が言及されているにもかかわらず、担当ファイルテーブルに含まれていない欠落が evaluator により発見された。L1 がタスク定義時に「Step N で言及される全ファイルが担当ファイルテーブルに含まれること」を完了条件に追加するパターンが有効。

## 提案
- `02_tasks.md` テンプレートまたは l1-manager のタスク作成ガイドに、スキルファイル変更時の固定チェック項目として追加する
- または `.claude/skills/l1-manager/SKILL.md` の「タスクリストに含める固定タスク」セクションに追加する
