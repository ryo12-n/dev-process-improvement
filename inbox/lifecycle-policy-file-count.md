# §1.4.3 tree 図にファイル総数を明記する提案

## 概要

`session-lifecycle-policy` §1.4.3 の tree 図にファイル総数（現在24ファイル）を明記することで、テンプレート移行施策の評価効率が向上する。

## 背景

- 評価者がテンプレート構造の一致を検証する際、tree 図との目視比較に加えてファイル数での数量チェックができると効率的
- 現状は tree 図のみで数量は明記されていないため、評価者が自分でカウントする必要がある

## 発見元

- L2-evaluator（initiative-template-2phase-migration SC-1 検証時）
