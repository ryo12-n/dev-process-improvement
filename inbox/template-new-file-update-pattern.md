# テンプレート新設時の連動更新パターンの固定タスク化

## 概要
initiative テンプレートに新ファイルを追加する場合、以下の5ステップが定型パターンとして必要になることが判明した。l1-manager の固定タスクとして追加することで、漏れを防止できる可能性がある。

## 5ステップパターン
1. テンプレートファイル作成
2. `l1-manager/SKILL.md` の初期化手順に追加
3. `l1-manager/SKILL.md` の関連ファイル一覧に追加
4. `docs/workflow.md` の該当セクション更新
5. ファイルオーナーシップテーブル更新

## 発見元
`dispatch-log-and-cost-tracking` 施策の L2-worker 知見（ルール化候補#1）、L2-evaluator 知見（ルール化候補#1）
