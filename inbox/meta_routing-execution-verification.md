# gate_review ルーティング実行確認の実在確認ステップ強化

## 発見元
メタ認知セッション 2026-03-11 / MC-002, MC-005

## 概要
gate_review の横展開セクションに「ルーティング実行済みであることを確認した」チェックボックスがあるが、実際には55件のアクショナブル宣言のうち12件（22%）が未実行。自己申告チェックが形骸化している。

## 提案
- l1-manager のクローズ手順で、ルーティング実行確認を「自己申告チェック」から「実在確認（ファイル・ISS-IDの存在を確認）」に強化する
- 具体的には: 各ルーティング先にファイル/エントリが存在することを grep 等で確認するステップを追加
